# 🚀 Nexus Landing Page

A modern and responsive landing page built using **HTML, CSS, and JavaScript**. The project features a clean dark-themed UI, smooth animations, interactive navigation, responsive design, and a contact form section.

## 🌐 Live Demo

https://nexus-landing-pages.onrender.com

---

## 📌 Features

- Modern dark-themed user interface
- Fully responsive design for mobile, tablet, and desktop
- Smooth scrolling navigation
- Interactive hamburger menu
- Animated hero section
- Feature showcase cards
- About section with statistics
- Contact form with validation
- Scroll reveal animations
- Footer with navigation links
- Optimized layout and performance

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Google Fonts
